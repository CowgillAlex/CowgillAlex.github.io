3D Voxel Engine

Published on: February 18, 2024

This is a 3D voxel engine, written in JavaScript using the ThreeJS Framework. It features features such as an orbit controls camera and procedural world generation with Simplex Noise.

It uses lil-gui to render the GUI to control the paramaters of the world generation

While adjusting the paramaters of the world generation, the performance is extremely unstable, especially when the world size is much larger. It normally runs at around the monitors framerate, as that seems to be what caps it.
