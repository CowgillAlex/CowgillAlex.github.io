24w19a is the second snapshot here. My stats say that I have been testing this project for over 30 hours since January this year O_O


Additions

  • WASD as an alternative to the arrow keys
  • Space key to jump

Changes

  • Linked the GRID list for exporting to the "i" (debug) key
  • Removed the player heart HUD: it is not used, there are no damage types yet
  • Removed quick move (a, d, f, g, h, j) to allow for WASD key movements
  • Limited Camera movement (player is on screen at all times)
  • Changed the way the columns are generated, to be less glitchy when going into the negative x axis

Bugs Fixed

  • When the player moves to the left, the player doesnt move smoothly. [SE-1]
  • When player jumps, after crawling, the player rotates, rather than turns
  • When the player is crawling, the hand isnt positioned correctly
  • Player can escape the world
